# Générateur de Quiz Interactif mAI

## Introduction
Le générateur de quiz mAI permet de créer automatiquement des quiz personnalisés à partir de n'importe quel contenu. Idéal pour la formation, l'évaluation et la mémorisation.

---

## Étape 1 : Préparer votre contenu

Le générateur accepte plusieurs formats sources :
- Texte brut / Markdown
- Documents PDF, Word, PowerPoint
- Pages web (URL)
- Transcriptions audio/vidéo
- Notes de canevas existantes

> ✅ Taille maximale : 50 000 mots par quiz

---

## Étape 2 : Configurer les paramètres du quiz

### Types de questions disponibles

| Type | Description | Meilleur usage |
|------|-------------|----------------|
| QCM | Questions à choix multiples | Évaluation générale |
| Vrai/Faux | Questions binaires | Vérification rapide |
| Rédaction | Questions ouvertes | Évaluation profonde |
| Association | Coupler éléments | Concepts reliés |
| Ordre | Mettre dans le bon ordre | Processus et séquences |

---

## Étape 3 : Définir le niveau de difficulté

1. **Débutant** : Questions basiques, indices disponibles
2. **Intermédiaire** : Questions nuancées, 1 indice maximum
3. **Avancé** : Questions complexes, aucun indice
4. **Expert** : Questions pièges, temps limité

> 💡 Astuce : Utilisez le niveau progressif pour des formations longues.

---

## Étape 4 : Générer et personnaliser

1. Importez votre contenu
2. Sélectionnez le nombre de questions (5-100)
3. Choisissez les types de questions
4. Cliquez sur **"Générer le quiz"**
5. Patientez 10-30 secondes selon la taille

### Options de personnalisation :
- Ajouter des explications pour chaque réponse
- Définir un temps limite par question
- Activer le mode aléatoire
- Personnaliser le message de fin
- Ajouter des badges de réussite

---

## Étape 5 : Partager et analyser

### Méthodes de partage :
1. **Lien public** - Accessible à tous
2. **Lien protégé** - Mot de passe requis
3. **Équipe uniquement** - Membres de votre espace
4. **Intégration** - Iframe pour votre site

### Tableau de bord analytique :
- Taux de réussite global
- Temps moyen par question
- Questions les plus ratées
- Progression individuelle
- Export des résultats en CSV/PDF

---

## Bonnes pratiques

1. Limitez les quiz à 20 questions maximum pour maintenir l'attention
2. Ajoutez toujours des explications pour les réponses
3. Testez votre quiz avant de le partager
4. Utilisez des niveaux progressifs pour les formations
5. Analysez les résultats pour améliorer votre contenu

---

## Exemples d'utilisation

- Évaluation post-formation
- Auto-formation sur un nouveau sujet
- Jeu d'équipe pendant les réunions
- Contrôle des connaissances avant certification
- Révision avant un examen
