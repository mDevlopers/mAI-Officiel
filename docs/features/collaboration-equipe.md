# Collaboration en Équipe sur mAI

## Introduction
mAI est conçu pour le travail d'équipe. Ce guide explique comment inviter des membres, gérer les permissions et collaborer efficacement sur les projets IA.

---

## Étape 1 : Créer et configurer votre équipe

1. Allez dans **"Paramètres" → "Équipe"**
2. Cliquez sur **"Créer une équipe"**
3. Remplissez les informations :
   - Nom de l'équipe
   - Description
   - Logo (optionnel)
4. Choisissez le plan adapté à votre taille

> 💡 Vous pouvez appartenir à plusieurs équipes en même temps.

---

## Étape 2 : Inviter des membres

### Méthodes d'invitation :

1. **Par email** : Entrez les adresses email des personnes à inviter
2. **Par lien d'invitation** : Générez un lien partageable (valide 7 jours)
3. **Par domaine** : Acceptez automatiquement tous les utilisateurs de votre domaine entreprise

### Rôles et permissions

| Rôle | Permissions |
|------|-------------|
| 🟢 Propriétaire | Tous les droits, peut supprimer l'équipe |
| 🟡 Administrateur | Gérer les membres, les factures, les paramètres |
| 🔵 Éditeur | Créer, modifier, supprimer du contenu |
| 🟢 Commentateur | Voir et commenter, pas de modification |
| ⚪ Lecteur | Voir seulement, aucune modification |

> ⚠️ Soyez prudent avec le rôle Propriétaire - il ne peut pas être révoqué par d'autres.

---

## Étape 3 : Partager des ressources

### Niveaux de partage :

1. **Équipe entière** : Tous les membres voient la ressource
2. **Groupe spécifique** : Seuls les membres d'un groupe y ont accès
3. **Utilisateurs sélectionnés** : Partage individuel
4. **Lien public** : Accessible à toute personne ayant le lien

### Comment partager un canevas :
1. Ouvrez le canevas
2. Cliquez sur **"Partager"** en haut à droite
3. Choisissez le niveau d'accès
4. Ajoutez les personnes ou groupes
5. Optionnel : Ajoutez un message personnalisé

---

## Étape 4 : Collaborer en temps réel

### Fonctionnalités de collaboration :

✅ **Édition simultanée** : Plusieurs personnes peuvent modifier le même canevas en même temps
✅ **Curseurs des collaborateurs** : Voyez ce que les autres sont en train de modifier
✅ **Commentaires** : Ajoutez des commentaires sur des sections spécifiques
✅ **Mentions** : Taggez des collègues avec `@nom` pour les notifier
✅ **Mode suggestion** : Proposez des modifications plutôt que de modifier directement
✅ **Chat intégré** : Discutez sans quitter le canevas

---

## Étape 5 : Gérer les groupes

Pour les grandes équipes, organisez vos membres en groupes :

1. Allez dans **"Équipe" → "Groupes"**
2. Créez un groupe (ex: "Marketing", "Développement")
3. Ajoutez les membres concernés
4. Partagez des ressources directement au groupe

### Exemples de groupes utiles :
- Direction
- Développement
- Marketing
- Support client
- Ressources Humaines

---

## Bonnes pratiques de collaboration

### ✅ Ce qu'il faut faire
1. Utilisez les commentaires pour les discussions
2. Mentionnez les personnes concernées pour les notifier
3. Activez le mode suggestion pour les modifications importantes
4. Documentez les décisions directement dans le canevas
5. Archivez les projets terminés

### ❌ Ce qu'il faut éviter
1. Modifier le travail de quelqu'un sans prévenir
2. Partager des ressources sensibles avec toute l'équipe
3. Garder des anciennes versions actives
4. Oublier de notifier les membres des modifications importantes

---

## Notifications et suivi

### Types de notifications :

| Canal | Quand |
|-------|-------|
| 🔔 Dans l'application | Vous êtes mentionné, quelqu'un commente |
| 📧 Email | Invitations, modifications importantes |
| 💬 Slack/Teams | Activité dans les projets suivis |
| 📱 Mobile | Notifications push (si application installée) |

### Personnalisez vos notifications :
Allez dans **"Paramètres" → "Notifications"** pour choisir ce que vous voulez recevoir et par quel canal.

---

## Audit et historique

### Journal d'activité complet :
Toutes les actions sont enregistrées :
- Qui a créé/modifié/supprimé quoi
- Date et heure exacte
- Adresse IP
- Type d'action

### Conservation :
- Gratuit : 30 jours d'historique
- Pro : 1 an d'historique
- Entreprise : Historique illimité
