# Mode Entraînement mAI

## Introduction
Le Mode Entraînement transforme mAI en un formateur personnel adaptatif. Il permet d'apprendre n'importe quel sujet à votre rythme, avec des exercices personnalisés et un suivi de progression.

---

## Comment ça marche

Le Mode Entraînement utilise l'IA pour :
1. Évaluer votre niveau initial
2. Créer un programme personnalisé
3. Adapter la difficulté en temps réel
4. Identifier vos points faibles
5. Mesurer votre progression

> 🎯 Parfait pour apprendre une nouvelle technologie, préparer un examen ou monter en compétences.

---

## Étape 1 : Démarrer un entraînement

1. Ouvrez un canevas contenant le contenu à apprendre
2. Cliquez sur **"Mode Entraînement"** dans la barre supérieure
3. Choisissez votre objectif :
   - 📚 Comprendre le contenu
   - ✍️ Savoir expliquer
   - 🎯 Maîtriser parfaitement
   - 🧠 Mémoriser à long terme
4. Définissez votre durée quotidienne souhaitée
5. Cliquez sur **"Commencer"**

---

## Étape 2 : Évaluation initiale

Avant de commencer, l'IA va évaluer votre niveau actuel :

1. 5 à 10 questions sur le sujet
2. Mélange de QCM et questions ouvertes
3. Durée : 5-10 minutes
4. Résultat : Niveau estimé et points à travailler

> 💡 Vous pouvez sauter l'évaluation si vous connaissez déjà votre niveau.

---

## Étape 3 : Programmes d'entraînement

### Types d'exercices disponibles :

| Type | Description | Meilleur pour |
|------|-------------|---------------|
| 📖 Résumé interactif | Explique chaque concept étape par étape | Découverte |
| ❓ Questions flash | Système de répétition espacée | Mémorisation |
| ✍️ Exercices pratiques | Mettez en pratique ce que vous apprenez | Maîtrise |
| 🎯 Cas pratiques | Résolvez des problèmes concrets | Application |
| 🗣️ Explication orale | Expliquez le concept à l'IA | Compréhension profonde |

---

## Étape 4 : Adaptation en temps réel

L'IA ajuste automatiquement la difficulté :

```
SI vous répondez correctement à 3 questions d'affilée
   → Augmente la difficulté
   → Introduit de nouveaux concepts

SI vous faites 2 erreurs sur le même sujet
   → Revient sur les bases
   → Explique le concept différemment
   → Propose des exercices supplémentaires
```

---

## Étape 5 : Suivi de progression

### Tableau de bord d'entraînement :

- 📊 Progression globale en pourcentage
- 📈 Courbe d'apprentissage sur 7/30/90 jours
- 🎯 Points maîtrisés vs points à travailler
- ⏱️ Temps passé par sujet
- 🏆 Badges et récompenses

### Métriques clés :
- **Taux de réussite** : Pourcentage de bonnes réponses
- **Vitesse** : Temps moyen par question
- **Rétention** : Ce que vous retenez après 7 jours
- **Progression** : Vitesse d'apprentissage

---

## Étape 6 : Fonctionnalités avancées

### 🔄 Répétition espacée
mAI utilise un algorithme scientifique pour vous revoir les concepts exactement au moment où vous êtes sur le point de l'oublier. Maximise votre rétention à long terme.

### 🎯 Ciblage des points faibles
L'IA identifie automatiquement les concepts que vous maîtrisez mal et crée des exercices ciblés pour les travailler.

### 📅 Rappels intelligents
Recevez des notifications pour vous entraîner au moment où vous êtes le plus réceptif, selon vos habitudes.

---

## Bonnes pratiques

### ✅ Pour progresser vite
1. Entraînez-vous tous les jours, même 10 minutes
2. Ne passez pas au niveau suivant tant que vous n'avez pas 80% de réussite
3. Utilisez les exercices d'explication orale pour valider votre compréhension
4. Revoyez régulièrement les points faibles
5. Faites des pauses toutes les 25 minutes

### ❌ Erreurs à éviter
1. Ne sautez pas l'évaluation initiale
2. Ne cherchez pas les réponses sur internet
3. Ne passez pas 2 heures d'affilée sans pause
4. Ne négligez pas les concepts difficiles

## Intégration avec les Quiz

Le Mode Entraînement est parfaitement intégré avec le Générateur de Quiz :
- Les questions ratées dans les quiz sont automatiquement ajoutées à votre entraînement
- Vous pouvez générer un quiz final pour valider votre formation
- Les résultats des quiz mettent à jour votre niveau automatiquement

---

## Limites

- Maximum 3 programmes d'entraînement actifs simultanément (Gratuit)
- Illimité (Pro et Entreprise)
- Taille maximum du contenu source : 100 000 mots
