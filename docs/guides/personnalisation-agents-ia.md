# Personnalisation des Agents IA mAI

## Introduction
mAI vous permet de créer des agents IA entièrement personnalisés, adaptés à vos cas d'usage spécifiques. Chaque agent peut avoir sa propre personnalité, ses connaissances et ses capacités.

---

## Étape 1 : Créer un nouvel agent

1. Allez dans la section **"Agents"** du menu principal
2. Cliquez sur **"+ Créer un agent"**
3. Donnez un nom et une description claire
4. Choisissez le modèle de base :
   - GPT-4o (polyvalent)
   - Claude 3 Opus (raisonnement long)
   - Gemini Advanced (multimodal)
   - Llama 3 (open source)

---

## Étape 2 : Définir la personnalité

### Paramètres de personnalité

| Paramètre | Effet | Plage |
|-----------|-------|-------|
| Créativité | Niveau d'originalité | 0-100 |
| Formalité | Ton des réponses | 0-100 |
| Verbosité | Longueur des réponses | 0-100 |
| Agressivité | Niveau de persuasion | 0-100 |
| Optimisme | Ton général | 0-100 |

### Exemple de configuration :
```
Agent : Assistant Support Client
Créativité : 30
Formalité : 70
Verbosité : 50
Agressivité : 10
Optimisme : 80
```

---

## Étape 3 : Ajouter des connaissances spécifiques

Votre agent peut intégrer plusieurs sources de connaissances :

1. **Documents uploadés** - PDF, Word, Excel
2. **Base de connaissances** - Articles et FAQ
3. **URLs externes** - Sites web et documentation
4. **Canevas existants** - Projets précédents
5. **Instructions manuelles** - Règles spécifiques

> 🔄 Les connaissances sont mises à jour automatiquement tous les 24h.

---

## Étape 4 : Configurer les capacités

Activez ou désactivez les fonctionnalités :

✅ Rédaction de texte  
✅ Analyse de données  
✅ Génération d'images  
✅ Traitement audio  
✅ Traduction automatique  
✅ Intégration API  
✅ Génération de code  
✅ Création de diagrammes

---

## Étape 5 : Définir les règles de comportement

Utilisez le champ **"Instructions système"** pour définir des règles strictes :

```
Tu es un agent de support client pour mAI.
- Réponds toujours en français
- Sois poli et empathique
- Ne donne jamais d'informations confidentielles
- Si tu ne sais pas, dis-le clairement
- Dirige vers un humain pour les problèmes complexes
- Termine chaque conversation par "Avez-vous besoin d'autre chose ?"
```

---

## Étape 6 : Tester et itérer

1. Utilisez l'onglet **"Test"** pour dialoguer avec votre agent
2. Testez des cas limites et des questions difficiles
3. Ajustez les paramètres selon les résultats
4. Demandez à vos collègues de tester également
5. Itérez jusqu'à obtenir le comportement souhaité

---

## Bonnes pratiques

1. Commencez simple et ajoutez des capacités progressivement
2. Testez systématiquement après chaque modification
3. Documentez les règles pour votre équipe
4. Mettez à jour les connaissances régulièrement
5. Utilisez les versions pour conserver les configurations fonctionnelles

---
