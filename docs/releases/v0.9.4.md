# mAI v0.9.4 — Thinking, sources transparentes et productivité

_Date de publication : 13 avril 2026._

## 🧠 Thinking Mode (Mode Raisonnement)

- Nouveau contrôle de profondeur de raisonnement depuis la barre de chat.
- Niveaux disponibles :
  - ⚡ **Rapide**
  - ⚖️ **Standard**
  - 🔥 **Approfondi**
  - 🚀 **Extrême**
- Conçu pour les prompts complexes (code, maths, analyse).

## 📊 Citations et sources

- Les réponses peuvent intégrer des références vérifiables.
- Objectif : améliorer la transparence et la confiance utilisateur.
- Les sources peuvent être consultées rapidement dans l’interface.

## 💬 Slash Commands (workflow power user)

- Menu slash plus intelligent, filtrage amélioré (préfixe, texte partiel, accents).
- Nouvelles commandes natives :
  - `/résume`
  - `/code`
  - `/quiz`
- Support des commandes personnalisées via stockage local, limité à **8 raccourcis**.

## 🎓 Apprentissage interactif

- La génération de quiz interactifs est renforcée dans la barre de chat.
- Les modules pédagogiques gardent une interaction directe et orientée compréhension.

## 🛠️ Correctifs majeurs

- Stabilisation de la navigation des commandes slash au clavier.
- Correction d’un cas limite où l’index de sélection devenait invalide si la liste filtrée changeait.
- Durcissement du parsing des commandes personnalisées (validation stricte + fallback sûr).

## 🎨 UX / UI (Liquid Glass)

- Le menu slash adopte une variante **Liquid Glass** cohérente avec le reste de l’app.
- Meilleure lisibilité en mode clair/sombre.

## ⚡ Performance

- Filtrage des commandes factorisé et mémorisé.
- Réduction des recalculs inutiles côté input.
